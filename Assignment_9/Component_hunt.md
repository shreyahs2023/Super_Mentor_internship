# Assignment: Component Hunt  
## Website Chosen: Netflix  

---

## Work Summary

1. Analyzed the Netflix website to understand its user interface structure and layout design.  
2. Identified key components such as the header, banner section, content rows, and footer.  
3. Broke down the homepage into reusable UI components like movie cards, navigation menus, and category sections.  
4. Sketched the hierarchical structure of components to visualize how the webpage is organized.  
5. Documented each component and its function in contributing to the overall user experience.

---

## Learning Outcomes

1. Developed the ability to break down complex websites into smaller, manageable components.  
2. Learned how modern streaming platforms use reusable UI elements for scalability.  
3. Gained understanding of component-based architecture used in frontend frameworks.  
4. Recognized how structured layouts improve maintainability and development efficiency.  
5. Understood the importance of planning UI components before implementation.
