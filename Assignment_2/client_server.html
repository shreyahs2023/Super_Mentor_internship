Client-Server Architecture

Client (Browser)
      |
      | HTTP Request
      v
Internet (DNS + Routing)
      |
      v
Server (Processes Request)
      |
      | HTTP Response
      v
Client (Displays Page)

Steps

  1. User enters URL
  2. DNS resolves IP
  3. Request sent to server
  4. Server processes request
  5. Response returned
  6. Browser renders page
